About

The goals of the app is to find the way of a web app develompment. Usually for every CRUD app needs to find solution for common tasks kind of show, edit, create and delete data, show spin and error messages. The project is focused on front-end develompent. The back-end of the app is mocked-up. In the app is presented ideas how the task may be done with cutting age technologies.

Technologies

The heard of the app is the terrible awsome angularjs. Design is build appon Bootstrap from twitter. Got a lot of ideas from AngularStrap, AngularUI, UI Bootstrap.

How to use

Here is no documentation yet. By the way here are few key components. There are crud controller and part/reveal directives. If you want to know more about the app, I will be happy to answer questions via GitHub.

Show rows
<table ng-controller='InitCtrl'>
	<tr ng-repeat='model in modelsList'>
		<td>{{model.name}}</td>
		<td>{{model.city}}</td>
	</tr>
</table>
Busy spin
<div ng-show="isBusy" class="well">LOADING...</div>
<table ng-hide='isBusy' ng-controller='InitCtrl'>
	<tr ng-repeat='model in modelsList'>
		<td>{{model.name}}</td>
		<td>{{model.city}}</td>
	</tr>
</table>
New and edit row
<table ng-controller='InitCtrl'>
	<!-- It needs part attribute and from controller -->
	<tr part='NewOrEdit' is='row' part-controller='NewOrEditFormCtrl'>
		<td colspan=3>
			<form ng-submit='createOrUpdate()'>
				<input ng-model='model.name'/>
				<!-- other attributes here -->
			</form>
		</td>
	</tr>
	
	<!-- It needs show controller and reveal attribute -->
	<tr ng-repeat='model in modelsList' ng-controller='ShowFormCtrl' reveal on='mode'>
		<td>{{model.name}}</td>
		<td>{{model.city}}</td>
		<td>
			<!-- Edit button is here -->
			<button ng-click='edit()'>Edit</button>
		</td>
	</tr>
</table>
